### Estrutura do Projeto:

`backend`: código do backend
`frontend`: código do frontend

# Como rodar o projeto?

criar arquivo .env com as seguintes informações:

```ini
BACKEND_PORT = 3001
```

## Para rodar o projeto, basta executar os comandos abaixo:

```bash
    npm install
    npm run dev
```
